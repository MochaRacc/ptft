Come name a tram for us.

Transnet is allowing you to name 6027, when the V Types start entering service.

 

Why name a specific tram?

Transnet is planning on naming new V Type tram #6027, as apart of the naming tradition of trams, like C Type 215 "Ian St. Albansbury" and D Type 516 "Arthur J. Monash", we're allowing the community

to name the latest tram name in production for the tram network.

 

When does the competition end?

June 27th, 2025 @ 2:30PM

 

What are the competition rules?

1. One response per person

2. The name should be appropriate for public viewing

3. The name suggestion must not contain brand names

4. Name suggestions must not be of living people without prior consent

5. Suggestions containing offensive language, hate speech or anything that does not appear to the general public will be removed.

6. Suggesting the same name continuously does not give the name a likely chance to be implimented, duplicate names will be merged into one

7. Not all suggestions will be make the final cut, if you made a suggestion and it wasn't used, please do not harass Transnet staff, the InfoCentre/InfoLine or it's benefactors.

8. The name will be announced by June 30th, and will enter service in the new name by that date
